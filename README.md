# ☁️ Data Redundancy Removal System

A **Flask-based Data Redundancy Removal System** developed as **Task 01** for the **CodeAlpha Cloud Computing Internship**.

The project demonstrates how duplicate records can be identified and removed to improve data quality, optimize storage, and maintain consistency in a cloud-based environment.

---

## 🚀 Features

* Add and manage records
* Detect duplicate entries
* Remove redundant data
* Responsive web interface
* SQLite database integration
* Simple and user-friendly design

---

## 🛠️ Tech Stack

* Python
* Flask
* SQLite
* HTML5
* CSS3
* Bootstrap

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/CodeAlpha_Task01_Data_Redundancy_Removal.git
```

Navigate to the project folder:

```bash
cd CodeAlpha_Task01_Data_Redundancy_Removal
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
