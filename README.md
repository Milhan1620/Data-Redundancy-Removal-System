# 📊 Data Redundancy Removal System

A Flask-based web application developed as **Task1** for the **CodeAlpha Cloud Computing Internship**.

This project helps identify and manage duplicate records, improving data quality, consistency, and storage efficiency.

---

## 🚀 Features

- Add new records
- View stored records
- Detect duplicate entries
- Remove redundant data
- Clean and responsive user interface
- SQLite database integration

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- HTML5
- CSS3

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/CodeAlpha_Task1_Data_Redundancy_Removal_System.git

cd CodeAlpha_Task1_Data_Redundancy_Removal_System

pip install -r requirements.txt

python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
